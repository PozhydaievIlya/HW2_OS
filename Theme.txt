Задание №0
В пространстве заданы n материальных точек. С некоторого момента точка с
наименьшей массой исчезает, передавая свою массу ближайшей к ней точке. Так
продолжается до тех пор, пока не останется одна точка. Реализовать этот процесс и найти
оставшуюся точку.


Задание №4
Cоздайте аналог массива — списка (ArrayList) языка Java. Реализуйте следующую
функциональность:
1. добавление элемента в конец списка — метод add(item);
2. вставка элемента в середину списка — метод insert(index, item);
3. количество элементов в массиве — метод size();
4. удаление элементов по индексу — метод remove(index);
5. изменение значения существующего элемента — метод set(index, item);
6. получение значения заданного элемента — метод get(index);
